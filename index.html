What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. Continuous Integration automatically checks and tests code after every push. Continuous Deployment automatically releases the updated version to users. This process improves software quality and speeds up development.

My Pipeline

When I push code to the main branch, GitHub Actions starts automatically. It checks that all required files exist. Then it uploads the website files and deploys them to GitHub Pages. Finally, the website updates automatically.
